# Реализованный функционал онлайн-калькулятора:

## 1. Расчёт конструкций и материалов
- Поддерживаемые типы изделий: окна, двери, перегородки, раздвижные лоджии.
- Профильные системы: холодный алюминий, тёплый алюминий, ПВХ (3-х и 5-и камерный).
- Ограничения профильных систем по типу изделия: ПВХ доступен только для окон и дверей; перегородки — только холодный алюминий; раздвижные лоджии — холодный или тёплый алюминий.
- Настраиваемые параметры: ширина, высота, количество.
- Опции остекления (для ПВХ): глухое, с открываемой створкой.
- Дополнительные модификации: покраска профиля RAL (для алюминиевых профилей), тонировка стёкол.

## 2. Логика ценообразования и расчёт услуг
- Мультипозиционность: возможность добавления неограниченного количества разных изделий в один заказ.
- Автоматический расчёт стоимости по площади и выбранному профилю с учётом минимальной расчётной площади (1 м² для окон, 2 м² для дверей).
- Учёт общих услуг на весь заказ: монтаж, демонтаж старых конструкций.
- Расчёт логистики: доставка с учётом удалённости объекта (в км).
- Итоговый результат: формирование предварительной вилки цен («от и до» с заложенной погрешностью ±5%).

## 3. Генерация Коммерческого предложения (КП в PDF)
- Автоматическое формирование документа в один клик.
- Брендирование: шапка с логотипом ООО «Комфорт+» и полными реквизитами (ИНН/КПП, юридический адрес, телефон, сайт, email).
- Индивидуальные схематичные чертежи: программная отрисовка схем для каждого типа конструкции (окно глухое, окно с поворотно-откидной створкой, дверь, перегородка, раздвижная лоджия) с нанесёнными размерами клиента.
- Сводная спецификация: таблица со всеми позициями, профилями, площадями и таблица сопутствующих работ.

## 4. Сбор и обработка заявок
- Форма сбора контактов: ФИО, телефон, название организации. Валидация введённых данных (проверка форматов и ограничений по размерам).
- Прямая интеграция с базой данных: отправка детальной заявки со всеми выбранными параметрами напрямую в систему (Supabase).

## 5. Техническая реализация
- Адаптивный веб-интерфейс (корректное отображение на мобильных устройствах и ПК).
- Готовый модуль для интеграции на основной сайт компании (через iframe с автоматической подстройкой высоты, чтобы избежать двойной прокрутки).

## 6. Динамическое управление ценами
- Все базовые цены (стоимость профилей, монтаж, демонтаж, доставка, тонировка, RAL-наценка) загружаются из базы данных.
- Изменение цен не требует изменения кода и пересборки приложения — выполняется через панель управления.
- При недоступности базы данных калькулятор продолжает работать с ценами по умолчанию.

# Реализованный функционал Дашборда (панели управления):

## 1. Единая база заявок
- Автоматический сбор всех заявок из калькулятора в единую таблицу в режиме реального времени (автообновление каждые 30 секунд).
- Сохранение полной истории: дата и время расчёта, контактные данные клиента (ФИО, телефон, компания).
- Детализация заказа: просмотр всех параметров, которые выбрал клиент (типы изделий, профили, точные размеры, площади, выбранные дополнительные опции и услуги).
- Фиксация итоговой расчётной стоимости по каждому обращению.

## 2. CRM-функционал и контроль
- Система статусов полного цикла сделки (10 этапов): Новая → В работе → Замер назначен → Замер выполнен → Согласование → Производство → Монтаж назначен → Монтаж выполнен → Завершён. Отдельный статус «Отказ» с возможностью возврата заявки в работу.
- Контролируемые переходы между статусами: для каждого этапа определён набор допустимых следующих статусов (исключает случайный перескок этапа).
- Контекстные формы при смене статуса — при переводе в каждый статус открывается специализированная форма для ввода необходимых данных:
  - «В работу» — итоговая сумма, адрес объекта, контрагент, номер договора, ответственный менеджер, тип отгрузки.
  - «Замер назначен» — дата и время замера, адрес, выбор замерщика из справочника сотрудников.
  - «Замер выполнен» — фиксация общей площади конструкций.
  - «Согласование» — номер счёта, итоговая сумма.
  - «Монтаж назначен» — дата и время монтажа.
  - «Отказ» — выбор причины из предустановленных вариантов (Дорого, Передумал, Выбрал конкурента, Не дозвонились, Не устроили сроки, Другое) с возможностью добавить комментарий.
- Развёрнутая карточка заказа с редактируемыми полями: менеджер, замерщик, контрагент, номер договора, номер счёта, тип отгрузки (самовывоз / доставка / монтаж), площадь, готовность цеха (%), сумма оплаченного.
- Справочник сотрудников (менеджеры, замерщики, монтажники, администраторы) — используется для назначения ответственных по заказу.
- Отслеживание оплаты: статус оплаты (Не оплачен / Частично / Оплачен) с фиксацией оплаченной суммы.
- Визуальный индикатор готовности цеха (0–100%) с полосой прогресса для контроля производства.
- Теги заказов: Горячий, Перезвонить, Ожидает замера, VIP — для быстрой визуальной приоритизации.
- Автоматический расчёт ориентировочных сроков изготовления (дни) и трудозатрат на монтаж (часы) по каждому заказу на основе типа профиля и вида изделия.

## 3. Два режима отображения заказов
- Режим «Список»: классическая таблица заказов с полной информацией, тегами, поиском по клиенту и фильтрацией по статусу.
- Режим «Канбан-доска»: визуальное представление заказов в виде колонок по статусам (9 колонок). Каждая колонка показывает количество заказов и суммарный бюджет. Быстрое перемещение заказа в следующий статус одной кнопкой.
- Переключение между режимами выполняется кнопками в шапке дашборда.

## 4. Календарь замеров и монтажей
- Визуальный месячный календарь с отметками назначенных замеров и монтажей.
- Отображение ответственных сотрудников (замерщиков/монтажников) и времени на каждую дату.
- Навигация по месяцам.

## 5. Карта объектов с маршрутизацией
- Интерактивная карта (Яндекс.Карты) с маркерами адресов объектов.
- Автоматическая оптимизация порядка объезда точек по времени замеров/монтажей или географической близости.
- Построение маршрута между точками.

## 6. Аналитика и графики
- Визуализация потока заявок (графики активности по дням/неделям).
- Сбор статистики: подсчёт общего количества лидов, количества заказов, количества отказов, суммарного бюджета, конверсии, среднего чека.
- Динамика конверсии: сравнение конверсии с предыдущим аналогичным периодом (тренд вверх/вниз).
- Воронка продаж: визуальное распределение заказов по этапам.
- Анализ спроса (наиболее часто выбираемые профили, услуги).
- Фильтрация по периодам: Сегодня / Неделя / Месяц / Всё время / Конкретная дата.

## 7. Аудит-лог (История действий)
- При каждом изменении статуса или полей заказа (сумма, адрес, менеджер и т.д.) в базу данных автоматически записывается действие.
- Фиксируется: тип действия, старое и новое значение, дата и время, email сотрудника, совершившего изменение.
- Панель истории доступна по кнопке с часами на каждом заказе.
- История является неизменяемым аудит-логом (только создание записей, без возможности удаления или редактирования).

## 8. Экспорт данных
- Экспорт в Excel (CSV): выгрузка текущего списка заказов с учётом выбранного периода. Поддержка кириллицы. Все CRM-поля (клиент, телефон, адрес, сумма, менеджер, контрагент, договор, счёт, оплата, статус, состав заказа, комментарий) распределены по столбцам.
- Экспорт в PDF: формирование печатного отчёта с итогами по суммам, оплатам и детальной таблицей заказов.

## 9. Администрирование и безопасность
- Изолированный доступ: закрытая веб-среда, доступная по прямой ссылке.
- Авторизация: защита панели индивидуальным логином и паролем для руководителя.
- Смена пароля: возможность изменить пароль из интерфейса дашборда.
- Управление ценообразованием: встроенная админ-панель для самостоятельного изменения базовых цен, стоимости материалов и услуг (монтаж, демонтаж, доставка, тонировка, RAL) без привлечения разработчиков.
- Управление нормативами сроков: настройка сроков изготовления и трудозатрат по каждому типу профиля и виду изделия.
